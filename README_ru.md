# Змейка

[English](README.md) | [简体中文](README_zh.md) | Русский
> 蛇行千里，始于一步。

### Обзор

Игра «Змейка», разработанная на C++ с использованием Visual Studio 2022  
<strong>Внешняя библиотека</strong>：[Raylib](https://www.raylib.com/), 
[Dear ImGui](https://github.com/ocornut/imgui), 
[rlImGui](https://github.com/raylib-extras/rlImGui)

### Особенности

- <strong>Два режима сложности. В сложном режиме генерируются случайные препятствия, а змея ускоряется по мере роста</strong>
- <strong>Эффекты мощного взрыва и дрожание экрана при столкновении</strong>
- <strong>Встроенный плеер (6 треков):  
  - Пробел: Пауза / Воспроизведение  
  - A / D: Предыдущий / Следующий трек  
  - W / S: Регулировка громкости (шаг ±10%)  
  (Примечание: Начальный уровень громкости оптимизирован в исходном коде)</strong>
- <strong>Наивысшие баллы в обоих режимах автоматически сохраняются в локальную папку</strong>
