# 贪吃蛇

[English](README.md) | [Русский](README_ru.md) | 简体中文
> 蛇行千里，始于一步。

### 项目概览

一个使用 C++ 开发的贪吃蛇游戏，开发环境为 Visual Studio 2022  
<strong>外部库</strong>：[Raylib](https://www.raylib.com/), 
[Dear ImGui](https://github.com/ocornut/imgui), 
[rlImGui](https://github.com/raylib-extras/rlImGui)

### 特性

- <strong>两种难度模式。困难模式随机生成障碍物且蛇的速度随着长度的增加而加快 </strong>
- <strong>碰撞时产生屏幕震动和爆炸效果 </strong>
- <strong>内置 6 首背景音乐，支持：
  - 空格键：暂停 / 继续
  - A / D：上一首 / 下一首
  - W / S：音量调节（±10%）  
  （注意：我已根据源文件调整初始音量）</strong>
