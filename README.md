# Snake Game

A Snake game developed in C++ using Visual Studio 2022  
External libraries used: Raylib, Dear ImGui and rlImGui

### Features

- Normal and Hard modes. In Hard mode random obstacles are generated and the snake speeds up as it grows
- Explosion effect when the snake dies by hitting an obstacle
- Sound effects for eating food and hitting obstacles
- Built-in 6 background tracks, supports:  
  - Press Space to pause/resume music  
  - Press A/D to switch to previous/next track  
  - Press W/S to increase/decrease volume by ±10% (initial volumes adjusted per source files)  
- Highest scores for both modes are automatically saved to a local folder
